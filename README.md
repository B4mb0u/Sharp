## Features
- Left / Right Clicker
- In-Game Overlay
- Advanced Randomization
- Hold / Always Mode
- Break Blocks
- RMB / LMB Lock
- BlockHit
- Shake Effect (JitterClick)
- Click Sounds
- Is in Menu Check
- Is Minecraft Focused Check
- Blatant Mode
- Recorder
- Auto Configuration Saver
- Hide GUI Toggle
- Always on Top Toggle
- Discord Rich Presence
